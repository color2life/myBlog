Dynamically Load Grunt Tasks

Matchdep is a node package which automatically load dependencies from 'package.json' file.

It automatically load GruntJS tasks, it happend to me a lot whenever I installed new GruntJS plugin I forget to load it as a task in the 'Gruntfile.js'

How to install

npm install --save matchdep
//then add next line to Gruntfile.js
require('matchdep').filterDev('grunt-*').forEach(grunt.loadNpmTasks);

This will look into 'package.json' file and load dev dependencies strating with 'grunt-' load all of them into 'Gruntfile.js' as tasks.
